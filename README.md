# 🧠 سیستم جعبه لایتنر

یک اپلیکیشن کامل برای یادگیری هوشمند با استفاده از Next.js 14, Prisma, و Tailwind CSS.

## 🚀️ ویژگی‌ها

- ✅ **احراز هویت:** با NextAuth.js و Prisma
- 🎯 **منطققه لایتنر:** پیاده‌سازی هوشمند برای مرور کارت‌ها
- 📚 **کتابخانه:** مجموعه‌های آماده کلمات (مثل IELTS)
- 📊 **مدیریت کارت‌ها:** افزودن، ویرایش و حذف کارت‌ها
- 📊 **آمار و نمودار:** نمودارهای تعاملی برای پیشرفت کاربر
- 🎨 **دیزاین مدرن:** رابط کاربری مدرن و واکنش‌پذیر
- 📱 **واکنش‌پذیر:** دیزاین واکنش‌پذیر با Tailwind CSS
- 🗄️ **آماده برای دیپلوی:** آماده برای دیپلوی روی Vercel با PostgreSQL

---

## 🚀️ شروع سریع

### پیش‌نیاز‌ها

1.  **Node.js و npm:** مطمئن شوید که آخرین نسخه‌های LTS را نصب کرده باشید.
2. **Git:** مطمئن شوید که Git نصب و پیکربندی‌های شما در GitHub هماهنگ است.

### 🗂️ راه‌اندازی

1.  **کلون این ریپازیتوری را در یک پوشه کپی کنید.**
2.  `git init` در ریشه پروژه اجرا کنید.
3.  `git add .` تمام فایل‌ها را اضافه کنید.
4.  `git commit -m "Initial commit: Add Leitner Box App"`
5.  به [GitHub](https://github.com) بروید و یک ریپازیتوری جدید بسازید.
6.  به Vercel بروید و پروژه را دیپلوی کنید.
